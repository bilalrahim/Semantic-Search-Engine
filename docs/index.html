Introduction Last updated: 2021-08-12

This document describes the REST APIs provided by netzoocloud. These are for researchers, experimentalists, biologists, etc who have a basic understanding of programming.

In this documentation, we will use Python to demo the API requests. The code can be used as is or as a template for your own modifications in Python or another language. We use the requests library, which can be installed with pip.

Implementation of the panda function API more about it here!
